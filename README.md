# GOAT-CORE-discord-bot
## installation.

1.you need the last version of node.js.

2.```npm i``` for installing the dependency.

3.creat a config.json.
``` 
{
    "prefix": "!" ,
    "token": "YOUR_TOKEN_HERE !"
} 
```
4.run ```npm start``` for stating the bot.
